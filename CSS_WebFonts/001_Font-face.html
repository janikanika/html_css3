В предыдущих версиях CSS разработчики были вынуждены использовать только те шрифты, которые гарантированы установлены на компьютере пользователя, в CSS3 разработчики могут использовать любые шрифты, которые они захотят. Когда вы найдете необходимый шрифт просто разместите его на веб-сервер и подключите его с помощью нового CSS3 свойствам @font-face. Подключенный шрифт будет скачан и отображен автоматически при загрузке страницы.
